03 — Material

Surfaces

Cream Card
Default surface. Warm white on cream page.

Ink Surface
Dark sections, CTAs, streak counters.

Electric Glow
Success, streak milestones, achievements.
linear-gradient(135deg, #E8FF3C, #00FF80)
Surface Rules
Warmth First
All neutral surfaces inherit warm cream (#FFF8EE). Never pure white for page backgrounds.
Gradient Sparingly
Pink→purple gradient is the signature moment. Max 1 per viewport.
Depth via Color
Depth through warm/cool contrast and subtle borders, not blur or transparency.

18 — Motion

Animation Specs

/* Attent entrance — stagger children */
@keyframes attent-enter {
  from { opacity: 0; transform: translateY(28px); }
  to { opacity: 1; transform: translateY(0); }
}
.card { animation: attent-enter 0.7s cubic-bezier(0.16, 1, 0.3, 1) both; }
.card:nth-child(2) { animation-delay: 0.08s; }

/* Float — Nova-style pop-out UI */
@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}

/* Streak rotation ring */
@property --ga { syntax: '<angle>'; initial-value: 0deg; inherits: false; }
@keyframes ring { from { --ga: 0deg; } to { --ga: 360deg; } }
