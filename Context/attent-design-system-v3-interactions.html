Gradient Glow Border

Cards that reveal a rotating conic gradient border on hover. Uses @property for CSS custom property animation.

/* Requires @property for animation */
@property --glow-angle {
  syntax: '<angle>';
  initial-value: 0deg;
  inherits: false;
}

.card::before {
  background: conic-gradient(from var(--glow-angle), #FFA2FF, #6D2EFF, #7DFFFF, #FFA2FF);
  animation: glowSpin 3s linear infinite;
}
